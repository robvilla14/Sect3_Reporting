# Section 3 Contractor Hours Dashboard

A browser-based tool for tracking contractor compliance with 
Section 3 worker hour requirements on HUD-funded public works projects.

## What It Does
Enter total employee hours, Section 3 worker hours, and Section 3 
targeted worker hours per contractor. The dashboard calculates 
project-level compliance rates, flags non-compliant entries, and 
generates a two-page PDF report with data tables and visual charts.

## Who It's For
Labor compliance specialists, housing agencies, and consultants 
managing Section 3 reporting obligations on federally funded projects.

## Features
- Multi-contractor data entry with live compliance validation
- Color-coded warning and error indicators per entry
- Summary stat cards with pass/fail status
- Four interactive charts: total hours, compliance overview, 
  detailed breakdown, and percentage trends
- Two-page PDF export (report data + visual analysis)
- CSV export for record-keeping

## Compliance Reference
Section 3 benchmarks per 24 CFR Part 75:
- Section 3 Workers: minimum 25% of total project labor hours
- Section 3 Targeted Workers: minimum 5% of total project labor hours

## Usage
Open in any modern browser. No installation or internet connection 
required after initial load. All processing happens locally in the browser.

## Export
PDF filename format: Section3_Report_[ProjectNumber]_[Date].pdf
CSV filename format: Section3_Report_[ProjectNumber]_[Date].csv

## Notes
This tool generates project-level compliance snapshots. For quarterly 
qualitative effort documentation, use the companion tool: 
Section 3 Quarterly Effort Report.
